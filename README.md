# CPMN-NeurIPS-2025

This repository contains the PyTorch implementation of CPMN for fully incomplete multi-view clustering.

## Overview

CPMN learns view-specific representations from disjoint views, aligns cluster prototypes across views, and refines clustering with prototype-aware contrastive learning.

## Requirements

- Python 3.x
- PyTorch
- NumPy
- SciPy
- scikit-learn
- munkres
